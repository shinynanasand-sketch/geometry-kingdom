## 1. 제품 개요

### 1.1 제품 설명

초등학생들이 AR(증강현실) 기술을 활용하여 기하학을 재미있고 직관적으로 학습할 수 있는 웹 기반 교육 애플리케이션입니다. 2D 평면 도형부터 3D 입체 도형까지, 그리고 통일신라시대의 주령구(14면체)까지 다양한 도형을 AR로 체험하며 학습합니다.

### 1.2 제품 가치 제안

- **시각화 학습**: AR을 통해 추상적인 도형 개념을 3D로 직접 보고 만질 수 있음
- **게임화 학습**: 퀴즈, 미니게임, 수집 요소로 학습 동기 부여
- **문화 융합**: 한국 전통 문화(주령구)와 수학 교육의 결합
- **자기주도 학습**: 레벨 자유 선택, 개인 맞춤 학습 속도
- **접근성**: 웹 기반으로 별도 앱 설치 없이 사용 가능

### 1.3 차별화 요소

1. **한국 최초** 주령구 기반 수학 교육 콘텐츠
2. **마커 기반 + 마커리스** 두 가지 AR 방식 지원
3. **초등 교육과정 100% 반영** (2015 개정 교육과정)
4. **STEAM 교육** 통합 (과학, 기술, 공학, 예술, 수학)
5. **오일러 공식** 등 고급 수학 개념까지 포함

## 2. 목표 및 비전

### 2.1 단기 목표 (3개월)

- 🎓 10개 초등학교 시범 운영
- 📊 사용자 1,000명 확보
- 🏆 교육 콘텐츠 공모전 출품

## 3. 타겟 사용자

### 3.1 주요 사용자 (Primary Users)

#### 초등학생 (1-6학년)
- **연령**: 7-13세
- **특성**: 
  - 디지털 네이티브 세대
  - 게임과 영상 콘텐츠에 익숙
  - 짧은 집중 시간 (10-15분)
  - 즉각적인 피드백 선호
- **니즈**:
  - 재미있는 학습 경험
  - 성취감과 보상
  - 친구들과 경쟁/협력
  - 시각적이고 직관적인 설명

### 3.2 부차 사용자 (Secondary Users)

#### 학부모
- **연령**: 30-50대
- **특성**:
  - 자녀 교육에 관심 많음
  - 스마트 기기 활용 가능
  - 안전하고 검증된 콘텐츠 선호
- **니즈**:
  - 자녀 학습 진도 확인
  - 교육적 가치 있는 콘텐츠
  - 유해 콘텐츠 차단
  - 학습 시간 관리

#### 초등학교 교사
- **연령**: 20-60대
- **특성**:
  - 교육과정 전문가
  - 다양한 교수법 활용
  - 학생 개별 맞춤 교육 추구
- **니즈**:
  - 교육과정 연계 자료
  - 학생 참여도 향상 도구
  - 평가 및 피드백 기능
  - 수업 활용 가이드

### 3.3 사용자 페르소나

#### 페르소나 1: 민준 (초등 3학년, 9세)
- **배경**: 게임을 좋아하고 수학은 어려워함
- **목표**: 재미있게 놀면서 수학 실력 향상
- **행동**: 매일 30분씩 게임하며 도형 카드 수집
- **불만**: 학교 수학 수업이 지루함

#### 페르소나 2: 서연 (초등 5학년, 11세)
- **배경**: 수학을 좋아하고 호기심이 많음
- **목표**: 어려운 문제에 도전하고 싶음
- **행동**: 모든 레벨 3성 클리어 목표
- **불만**: 쉬운 문제는 재미없음

#### 페르소나 3: 김 선생님 (초등교사, 35세)
- **배경**: 수학 교과 담당, 혁신적 교수법 관심
- **목표**: 학생들의 공간 지각 능력 향상
- **행동**: 수업 시간에 AR 활용, 학생 진도 체크
- **불만**: 기존 교구는 비싸고 관리 어려움

## 5. 사용자 스토리

### 5.1 학생 사용자 스토리

#### US-1: AR로 도형 보기
**As a** 초등학생  
**I want to** 마커를 카메라에 비춰서 3D 도형을 보고 싶다  
**So that** 도형이 어떻게 생겼는지 직접 확인할 수 있다

**인수 기준**:
- 마커를 비추면 3초 이내에 도형이 나타남
- 도형이 자동으로 회전함
- 도형 이름과 정보가 표시됨

#### US-2: 퀴즈 풀기
**As a** 초등학생  
**I want to** 도형에 대한 퀴즈를 풀고 싶다  
**So that** 내가 얼마나 알고 있는지 확인하고 점수를 받을 수 있다

**인수 기준**:
- 문제를 읽고 4개 선택지 중 선택
- 정답이면 "정답!" 메시지와 코인 획득
- 오답이면 "다시 도전!" 메시지와 힌트 제공

#### US-3: 도형 카드 수집
**As a** 초등학생  
**I want to** 도형 카드를 모으고 싶다  
**So that** 도감을 완성하고 친구들에게 자랑할 수 있다

**인수 기준**:
- 레벨 클리어 시 카드 획득
- 도감에서 수집한 카드 확인
- 진행률 표시 (예: 15/50)

#### US-4: 주령구 체험
**As a** 초등학생  
**I want to** 주령구를 AR로 보고 굴려보고 싶다  
**So that** 신라시대 문화재를 재미있게 배울 수 있다

**인수 기준**:
- 주령구 3D 모델이 AR로 표시됨
- 화면을 터치하면 주령구가 굴러감
- 나온 면에 따라 미션 수행

### 5.2 교사 사용자 스토리

#### US-5: 수업 활용
**As a** 초등학교 교사  
**I want to** 수업 시간에 AR 도형을 보여주고 싶다  
**So that** 학생들이 입체 도형을 쉽게 이해할 수 있다

**인수 기준**:
- 프로젝터나 대형 화면에 연결 가능
- 마커를 크게 인쇄하여 사용
- 학생들이 동시에 볼 수 있음

#### US-6: 학생 진도 확인
**As a** 초등학교 교사  
**I want to** 학생들의 학습 진도를 확인하고 싶다  
**So that** 개별 맞춤 지도를 할 수 있다

**인수 기준**:
- 학생별 완료 레벨 확인
- 정답률 및 취약 영역 파악
- 학습 시간 통계

### 5.3 학부모 사용자 스토리

#### US-7: 자녀 학습 모니터링
**As a** 학부모  
**I want to** 자녀가 얼마나 학습했는지 보고 싶다  
**So that** 학습 습관을 관리하고 격려할 수 있다

**인수 기준**:
- 일일/주간 학습 시간 확인
- 획득한 배지 및 성취도 확인

## 7. 제약사항 및 가정

### 7.1 기술적 제약사항

#### TC-1: 브라우저 호환성
- **제약**: AR.js는 WebRTC 지원 브라우저 필요
- **영향**: IE 11 이하 미지원
- **대응**: Chrome, Safari, Edge 권장 안내

#### TC-2: 카메라 권한
- **제약**: AR 사용 시 카메라 접근 권한 필수
- **영향**: 권한 거부 시 AR 기능 사용 불가
- **대응**: 권한 요청 안내 및 대체 학습 모드 제공

#### TC-3: 성능 제약
- **제약**: 저사양 기기에서 AR 성능 저하 가능
- **영향**: 프레임 드롭, 마커 인식 불안정
- **대응**: 최소 사양 안내, 2D 모드 제공

#### TC-4: 네트워크 의존성
- **제약**: CDN 라이브러리 로드 필요
- **영향**: 오프라인 사용 제한
- **대응**: 캐싱 전략, PWA 고려

### 7.2 비즈니스 제약사항

#### BC-1: 예산
- **제약**: 개인 프로젝트로 예산 제한
- **영향**: 유료 리소스 사용 불가
- **대응**: 무료 오픈소스 활용

#### BC-2: 개발 인력
- **제약**: 1인 개발
- **영향**: 개발 속도 제한
- **대응**: MVP 우선, 단계적 개발

#### BC-3: 법적 제약
- **제약**: 아동 개인정보 보호법 (COPPA)
- **영향**: 13세 미만 개인정보 수집 제한
- **대응**: LocalStorage만 사용, 서버 저장 없음

### 7.3 가정 (Assumptions)

#### A-1: 사용자 환경
- 사용자는 스마트폰 또는 태블릿 보유
- 카메라가 정상 작동하는 기기 사용
- 안정적인 인터넷 연결 (최소 3G)

#### A-2: 사용자 능력
- 기본적인 스마트 기기 조작 가능
- 한글 읽기 가능 (초등 1학년 수준)
- 부모 또는 교사의 초기 안내 가능

#### A-3: 교육 환경
- 학교에서 스마트 기기 사용 허용
- 교사가 AR 기술에 대한 기본 이해
- 마커 인쇄 가능한 환경

#### Phase 1–4 구현 요약 (2026-07-04)

- **기본 학습 모드**: 카메라 없이 3D로 즉시 시작 (권한 팝업 없음)

- **레벨 1 (기초)**: 삼각형, 사각형, 원
- **레벨 2 (심화)**: 정사각형, 직사각형, 마름모, 평행사변형, 사다리꼴, 직각·이등변삼각형
- **레벨 3–4**: 입체 기초/심화 도형, 배치·선택·크기(−/+ )·드래그 회전·삭제
- **퀴즈**: 레벨 1–7 문제 분리, `quiz-data.js`로 캐시 이슈 완화
