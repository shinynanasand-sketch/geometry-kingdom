## 📖 한국 초등 수학 교육과정 매핑

### 2015 개정 교육과정 기준

#### **1-2학년 (도형 영역)**

**[2수03-03]** 여러 가지 물건을 관찰하여 평면도형을 찾을 수 있다.
- **레벨 1 연계**: 주변에서 삼각형, 사각형, 원 찾기
- **AR 활동**: 교실, 집 안에서 도형 찾기 게임

**[2수03-04]** 삼각형, 사각형, 원을 알고 구별할 수 있다.
- **레벨 1 연계**: 도형의 변과 꼭짓점 세기
- **AR 활동**: 3D로 도형 관찰하며 특징 파악

#### **3-4학년 (도형 영역)**

**[4수03-01]** 직각을 알고, 직각과 비교하여 예각과 둔각을 이해한다.
- **레벨 2 연계**: 각도기 사용법, 각의 종류
- **AR 활동**: 도형을 회전하며 각도 측정

**[4수03-02]** 여러 가지 사각형을 알아본다.
- **레벨 2 연계**: 정사각형, 직사각형, 마름모, 평행사변형
- **AR 활동**: 사각형 분류 게임

**[4수03-05]** 선대칭도형을 이해한다.
- **레벨 2 연계**: 대칭축 찾기
- **AR 활동**: 도형을 접어보며 대칭 확인

**[4수03-06]** 직육면체를 알고, 그 성질을 이해한다.
- **레벨 3 연계**: 면, 모서리, 꼭짓점
- **AR 활동**: 직육면체 360도 관찰

**[4수03-07]** 직육면체의 겉넓이를 구할 수 있다.
- **레벨 5 연계**: 겉넓이 계산
- **AR 활동**: 전개도로 겉넓이 시각화

#### **5-6학년 (도형 영역)**

**[5수03-03]** 평행사변형, 삼각형, 마름모, 사다리꼴의 넓이를 구할 수 있다.
- **레벨 5 연계**: 넓이 공식 학습
- **AR 활동**: 도형에 치수 표시하며 계산

**[6수03-08]** 각기둥과 각뿔을 알고, 그 성질을 이해한다.
- **레벨 4 연계**: 각기둥, 각뿔의 구조
- **AR 활동**: 3D 모델로 구조 탐구

**[6수03-09]** 원기둥과 원뿔을 알고, 그 성질을 이해한다.
- **레벨 4 연계**: 원기둥, 원뿔의 특징
- **AR 활동**: 회전체 애니메이션

**[6수03-10]** 여러 가지 입체도형의 전개도를 그릴 수 있다.
- **레벨 4 연계**: 전개도 학습
- **AR 활동**: 전개도 접기 애니메이션

**[6수03-11]** 직육면체의 부피를 구할 수 있다.
- **레벨 5 연계**: 부피 공식
- **AR 활동**: 부피 시각화

**[6수03-12]** 원기둥의 겉넓이와 부피를 구할 수 있다.
- **레벨 5 연계**: 원기둥 계산
- **AR 활동**: 전개도와 부피 동시 표시

## 🎯 레벨별 학습 목표

### 레벨 1: 평면 도형 기초 ⭐

**대상 학년**: 1-2학년  
**교육과정**: [2수03-03], [2수03-04]

4. **평행사변형**: 마주 보는 변이 평행

### 레벨 5: 넓이와 부피 ⭐⭐⭐

**대상 학년**: 5-6학년  
**교육과정**: [5수03-03], [6수03-11], [6수03-12]

#### 학습 목표
1. 평면도형의 넓이를 구할 수 있다.
2. 입체도형의 부피를 구할 수 있다.
3. 실생활 문제에 적용할 수 있다.

#### 성취기준
- **기본**: 기본 공식을 사용할 수 있다.
- **우수**: 복합 도형의 넓이를 구할 수 있다.
- **최우수**: 실생활 문제를 해결할 수 있다.

#### 넓이 공식

| 도형 | 공식 |
|------|------|
| 직사각형 | 가로 × 세로 |
| 정사각형 | 한 변 × 한 변 |
| 평행사변형 | 밑변 × 높이 |
| 삼각형 | (밑변 × 높이) ÷ 2 |
| 사다리꼴 | (윗변 + 아랫변) × 높이 ÷ 2 |

#### 부피 공식

| 도형 | 공식 |
|------|------|
| 직육면체 | 가로 × 세로 × 높이 |
| 정육면체 | 한 모서리 × 한 모서리 × 한 모서리 |
| 원기둥 | (밑넓이) × 높이 |

## 🔬 과학적 설명 콘텐츠

### 1. 오일러 공식 (Euler's Formula)

#### 공식
```
V - E + F = 2
```
- V: 꼭짓점 (Vertices)
- E: 모서리 (Edges)
- F: 면 (Faces)

#### 초등학생용 설명
```
[오일러 박사]: "모든 입체도형에는 특별한 규칙이 있어!"
[네모]: "꼭짓점 - 모서리 + 면 = 2"
[삼이]: "정육면체로 확인해볼까?"

정육면체:
- 꼭짓점: 8개
- 모서리: 12개
- 면: 6개

계산: 8 - 12 + 6 = 2 ✓
```

### 2. 왜 정다면체는 5개뿐인가?

#### 조건
1. 모든 면이 같은 정다각형
2. 각 꼭짓점에 모이는 면의 개수가 같음
3. 꼭짓점에서 각도의 합 < 360°

#### 정삼각형으로 만들기
```
3개 모으면: 60° × 3 = 180° < 360° → 정사면체 ✓
4개 모으면: 60° × 4 = 240° < 360° → 정팔면체 ✓
5개 모으면: 60° × 5 = 300° < 360° → 정이십면체 ✓
6개 모으면: 60° × 6 = 360° = 360° → 평면 ❌
```

#### 정사각형으로 만들기
```
3개 모으면: 90° × 3 = 270° < 360° → 정육면체 ✓
4개 모으면: 90° × 4 = 360° = 360° → 평면 ❌
```

#### 정오각형으로 만들기
```
3개 모으면: 108° × 3 = 324° < 360° → 정십이면체 ✓
4개 모으면: 108° × 4 = 432° > 360° → 불가능 ❌
```

#### 정육각형 이상
```
정육각형: 120° × 3 = 360° = 360° → 평면 ❌
정칠각형: 128.57° × 3 = 385.71° > 360° → 불가능 ❌
```

**결론**: 정다면체는 5개뿐!

### 3. 주령구의 수학

#### 깎은 정육면체
```
정육면체의 8개 꼭짓점을 깎으면:
- 원래 정사각형 면 6개 → 그대로
- 깎인 꼭짓점 8개 → 정삼각형 8개 생성

```

```
V = 14 (꼭짓점)

V - E + F = 14 - 24 + 14 = 4

```
