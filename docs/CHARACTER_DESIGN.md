## 🌟 메인 캐릭터

### 1. 삼이 (Sam) 🔺

#### 기본 정보
- **도형**: 삼각형
- **색상**: 밝은 빨강 (#FF6B6B)
- **나이**: 8세 (초등학교 2학년)
- **성별**: 남자
- **성격**: 활발하고 호기심 많음, 에너지 넘침

#### 외모
- 정삼각형 모양의 몸
- 큰 눈 (반짝이는 눈동자)
- 웃는 얼굴
- 작은 팔다리
- 빨간 모자 착용 (선택사항)

#### 특징
- **장점**: 빠른 행동력, 긍정적, 친구들을 잘 이끔
- **단점**: 가끔 성급함, 생각보다 행동이 먼저
- **특기**: 빠르게 움직이기, 문제 발견하기
- **좋아하는 것**: 모험, 달리기, 새로운 것 발견
- **싫어하는 것**: 가만히 있기, 지루한 것

#### 역할
- 플레이어의 첫 번째 친구
- 레벨 1-2 주요 안내자
- 평면 도형 전문가

#### 등장 시점
- 메인 화면 (첫 등장)
- 레벨 1: 평면 도형 기초
- 레벨 2: 평면 도형 심화
- 미니게임: 도형 사냥, 도형 레이싱

- **성별**: 남자
- **성격**: 차분하고 논리적, 똑똑함

#### 외모

- 안경 착용 (둥근 안경)
- 진지한 표정 (가끔 미소)
- 책을 들고 있음 (선택사항)

#### 특징
- **장점**: 논리적 사고, 문제 해결 능력, 침착함
- **단점**: 가끔 너무 신중함, 완벽주의
- **특기**: 계산하기, 설명하기, 전략 세우기
- **좋아하는 것**: 책 읽기, 퍼즐, 수학
- **싫어하는 것**: 혼란, 무질서

#### 역할
- 지식 전달자
- 레벨 2, 5 주요 안내자
- 계산 및 공식 설명

#### 등장 시점
- 레벨 2: 평면 도형 심화
- 레벨 5: 넓이와 부피
- 퀴즈 모드 (힌트 제공)
- 미니게임: 도형 매칭, 도형 퍼즐

#### 특징

- **단점**: 가끔 너무 자신만만함
- **특기**: 3D 세계 안내, 입체 도형 설명
- **좋아하는 것**: 도전, 리더 역할, 멋진 것
- **싫어하는 것**: 실패, 약한 모습 보이기

#### 역할
- 입체 도형 전문가
- 레벨 3-4 주요 안내자
- 3D 세계 가이드

#### 등장 시점
- 레벨 3: 입체 도형 기초
- 레벨 4: 입체 도형 심화
- AR 모드 (3D 설명)
- 미니게임: 도형 쌓기

### 5. 스피어 (Sphere) 🌐

#### 기본 정보
- **도형**: 구
- **색상**: 은색/하늘색 (#87CEEB)
- **나이**: 불명 (신비로움)
- **성별**: 중성
- **성격**: 신비롭고 지혜로움, 차분함

#### 외모
- 구 모양의 몸
- 반짝이는 표면 (홀로그램 효과)
- 부드러운 미소
- 빛나는 오라
- 떠다니는 듯한 느낌

#### 특징
- **장점**: 지혜로움, 통찰력, 비밀 지식
- **단점**: 가끔 너무 신비로워서 이해하기 어려움
- **특기**: 힌트 제공, 비밀 발견, 특별한 지식
- **좋아하는 것**: 명상, 우주, 수학의 아름다움
- **싫어하는 것**: 서두름, 소란

#### 역할
- 비밀 힌트 제공자
- 특별 지식 전달
- 숨겨진 요소 안내

#### 등장 시점
- 어려운 문제 (힌트 제공)
- 숨겨진 배지 발견
- 레벨 6: 종합 챌린지
- 특별 이벤트

### "도형 왕국 구하기"

#### 프롤로그
```
옛날 옛적, 도형 왕국에는 모든 도형들이 행복하게 살고 있었어요.
삼각형, 사각형, 원, 그리고 입체 도형들까지...
모두가 조화롭게 지내며 수학의 아름다움을 즐겼답니다.

그런데 어느 날, 악당 헷갈이가 나타났어요!
헷갈이는 도형들을 뒤죽박죽 섞어버리고,
도형 왕국을 혼란에 빠뜨렸어요.

"히히히! 이제 아무도 도형을 구별할 수 없을 거야!"

도형 왕국의 왕은 용감한 모험가를 찾고 있어요.
바로 당신입니다!

삼이, 네모, 동글이와 함께
도형 왕국을 구해주세요!
```

**레벨 1: 평면 마을**

[삼이]: "안녕! 나는 삼이야. 도형 왕국에 온 걸 환영해!"

[삼이]: "삼각형, 사각형, 원을 찾아서 구해줘!"

[미션]: 평면 도형 찾기
[보상]: 평면 마을 해방
```

**레벨 2: 각도 숲**
```
[네모]: "평면 마을을 구해줘서 고마워!"
[네모]: "하지만 각도 숲에 더 큰 문제가 있어."
[네모]: "헷갈이가 모든 각도를 뒤섞어놨어."

[미션]: 각도 구별하기, 사각형 분류
