## 🎲 주령구란?

### 기본 정보

**주령구 (酒令具)**
- **한자**: 酒(술 주) + 令(명령 령) + 具(연장 구)
- **의미**: 술자리에서 사용하는 놀이 도구
- **시대**: 통일신라시대 (8-9세기)
- **발견**: 1975년 경주 안압지 (월지)
- **지정**: 국보 제74호
- **소장**: 국립경주박물관
- **재질**: 흑칠 (검은 옻칠)
- **크기**: 높이 약 5.4cm

### 특징

1. **세계 유일**: 가장 오래된 14면체 주사위
2. **완벽한 보존**: 1,300년이 지났지만 글자가 선명
3. **수학적 가치**: 깎은 정육면체 구조
4. **문화적 가치**: 신라 귀족 문화의 증거

## 📜 역사적 배경

### 통일신라시대 (676-935년)

#### 시대 상황
- **정치**: 삼국 통일 후 안정기
- **경제**: 무역 발달, 경제 번영
- **문화**: 불교 문화 전성기
- **사회**: 귀족 문화 발달

#### 안압지 (雁鴨池)
- **현재 명칭**: 월지 (月池)
- **건설**: 674년 (문무왕 14년)
- **용도**: 왕실 정원, 연회 장소
- **규모**: 동서 200m, 남북 180m
- **특징**: 인공 연못, 3개의 섬

### 발견 과정

#### 1975년 발굴
```
📅 1975년
📍 경주 안압지 바닥
🔍 발굴 조사 중 발견
💎 진흙 속에 1,300년간 보존
🏛️ 국립경주박물관으로 이관
🎖️ 1985년 국보 제74호 지정
```

#### 발견의 의미
- 신라 귀족들의 놀이 문화 증명
- 신라인들의 수학 지식 수준 확인
- 세계 최고(最古)의 14면체 주사위
- 한국 전통 놀이 문화 연구 자료

## 🔷 14면체 구조

### 기하학적 구조

#### 깎은 정육면체 (Truncated Cube)

```
정육면체 (Cube)
    ↓
8개 꼭짓점 깎기
    ↓
14면체 (Tetrakaidecahedron)
```

### 면의 구성

#### 14개 면
1. **정사각형 면**: 6개
   - 원래 정육면체의 6개 면
   - 크기가 약간 작아짐

2. **정삼각형 면**: 8개
   - 깎인 꼭짓점에서 생성
   - 8개 꼭짓점 → 8개 삼각형

### 구성 요소

| 요소 | 개수 | 설명 |
|------|------|------|
| 면 (Faces) | 14 | 정사각형 6개 + 정삼각형 8개 |
| 모서리 (Edges) | 24 | 정육면체 12개 + 깎기로 생성 12개 |
| 꼭짓점 (Vertices) | 14 | 각 모서리의 끝점 |

### 제작 과정

#### 1단계: 정육면체 준비
```
┌─────────┐
│         │
│    ■    │  정육면체
│         │
└─────────┘
```

#### 2단계: 꼭짓점 표시
```
    ●
   ╱ ╲
  ╱   ╲
 ●─────●  8개 꼭짓점 표시
```

#### 3단계: 꼭짓점 깎기
```
    ▽
   ╱ ╲
  ╱   ╲
 ▽─────▽  각 꼭짓점을 깎아냄
```

#### 4단계: 14면체 완성
```
  정삼각형
    ╱╲
   ╱  ╲
  ╱ 정 ╲  정사각형 + 정삼각형
 ╱ 사각╲  = 14면체
╱  형  ╲
```

## 📐 수학적 원리

### 오일러 공식 적용

#### 일반 볼록 다면체
```
V - E + F = 2

V: 꼭짓점 (Vertices)
E: 모서리 (Edges)
F: 면 (Faces)
```

#### 주령구의 경우
```
V = 14 (꼭짓점)
E = 24 (모서리)
F = 14 (면)

V - E + F = 14 - 24 + 14 = 4
```

**왜 2가 아닐까?**
- 주령구는 특수한 경우
- 깎은 정육면체는 준정다면체
- 오일러 공식의 변형 적용

### 준정다면체 (Archimedean Solid)

#### 정의
- 2가지 이상의 정다각형으로 구성
- 모든 꼭짓점이 동일한 구조
- 13가지 종류 존재

#### 주령구의 위치
- 13가지 준정다면체 중 하나
- 깎은 정육면체 (Truncated Cube)
- 정사각형 6개 + 정삼각형 8개

### 대칭성

#### 회전 대칭
- 3회 회전 대칭축: 4개
- 2회 회전 대칭축: 3개
- 4회 회전 대칭축: 6개

#### 반사 대칭
- 대칭면: 9개

### 게임 1: 주령구 모험 🗺️

#### 게임 설정
- **인원**: 1-4명
- **목표**: 가장 먼저 도착지 도달
- **시간**: 약 15-20분

#### 보드 구성
```
출발 → 경주 → 안압지 → 첨성대 → 불국사 → 석굴암 → 도착 🏁
  ↓      ↓       ↓        ↓        ↓        ↓
 10칸   10칸    10칸     10칸     10칸     10칸
```

#### 특별 칸
- 🏛️ **유적지 칸**: 역사 이야기 듣기 + 코인 10개
- 📐 **도형 칸**: 퀴즈 풀기 (정답 시 다시 굴림)
- 🎁 **보물 칸**: 코인 30개 획득
- 🎵 **놀이 칸**: 미니게임 (승리 시 2칸 전진)
- ⚡ **번개 칸**: 뒤로 2칸

#### 게임 진행
1. 순서 정하기 (주령구 굴려서 큰 숫자 순)
2. 주령구 굴리기
3. 나온 미션 수행
4. 특별 칸 효과 적용
5. 다음 플레이어 차례

#### 승리 조건
- 가장 먼저 도착지 도달
- 동시 도착 시 코인 많은 사람 승리
